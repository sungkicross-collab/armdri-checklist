Firebase 설정

학생들과 데이터를 공유하려면
Firebase 무료 계정이 필요합니다.
아래 단계를 따라 설정해 주세요.

1Firebase 프로젝트 만들기

console.firebase.google.com 접속
→ "프로젝트 추가" → 프로젝트 이름 입력 (예: armdri-checklist)
→ Google 애널리틱스 사용 안 함 → 프로젝트 만들기

2Realtime Database 활성화

왼쪽 메뉴 → "빌드" → "Realtime Database" → "데이터베이스 만들기"
→ 위치: asia-southeast1 선택 → 테스트 모드로 시작
→ 생성 후 나타나는 URL 복사 (예: https://xxx-default-rtdb.asia-southeast1.firebasedatabase.app)

3웹 앱 등록 후 API 키 복사

프로젝트 홈 → "앱 추가" → 웹(</>) 선택
→ 앱 닉네임 입력 → 등록
→ firebaseConfig 에서 apiKey와 databaseURL 복사

4아래에 입력하기

설정값은 이 기기에만 저장됩니다.
관리자 기기와 학생 기기 모두에서 동일하게 설정해 주세요.
